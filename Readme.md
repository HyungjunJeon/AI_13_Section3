# 음악 장르 추천 서비스 개발 프로젝트

코드스테이츠 AI 부트캠프 Section3 프로젝트입니다

## 프로젝트 소개

Melon Playlist 공유링크를 입력받아 Playlist의 앨범명, 곡명, 아티스트 명 데이터를 가져와 이를 기반으로 음악 장르를 추천해주는 서비스입니다   
Kakao가 제공한 Melon Playlist 데이터와 RandomForest 모델을 활용하였습니다
